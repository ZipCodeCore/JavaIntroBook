=== Multi-Dimensional Arrays

Multi-dimensional arrays are arrays that contain other arrays.
Think of them as grids, tables, or even cubes of data.
The most common type is a 2D array, which is like a spreadsheet with rows and columns.

==== What are 2D Arrays?

A 2D array is essentially an "array of arrays."
Imagine a tic-tac-toe board, a chess board, or a spreadsheet - these are all examples of 2D grids that we can represent with 2D arrays.

[source]
----
// A 2D array representing a 3x3 tic-tac-toe board
char[][] ticTacToe = {
    {'X', 'O', 'X'},
    {'O', 'X', 'O'},
    {'X', 'O', 'X'}
};
----

==== Creating 2D Arrays

There are several ways to create 2D arrays:

===== Method 1: Declare Size First

[source]
----
public class ZipCode {

  void compute() {
    // Create a 3x4 array (3 rows, 4 columns)
    int[][] numbers = new int[3][4];
    
    // Fill it with some values
    numbers[0][0] = 10;
    numbers[0][1] = 20;
    numbers[1][2] = 30;
    numbers[2][3] = 40;
    
    System.out.println("Value at row 1, column 2: " + numbers[1][2]);
  }

  public static void main(String[] args) { new ZipCode().compute(); }
}
----

===== Method 2: Initialize with Values

[source]
----
// Create and initialize a 2D array
int[][] matrix = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// This creates a 3x3 grid:
// 1 2 3
// 4 5 6
// 7 8 9
----

==== Accessing Elements

To access elements in a 2D array, you use two indices: `[row][column]`

[source]
----
public class ZipCode {

  void compute() {
    int[][] scores = {
        {85, 92, 78},    // Student 0's scores
        {90, 88, 95},    // Student 1's scores
        {76, 89, 82}     // Student 2's scores
    };
    
    // Get student 1's second test score (index [1][1])
    int score = scores[1][1];
    System.out.println("Student 1's second test: " + score);  // prints 88
    
    // Change student 0's first test score
    scores[0][0] = 90;
    System.out.println("Updated score: " + scores[0][0]);  // prints 90
  }

  public static void main(String[] args) { new ZipCode().compute(); }
}
----

==== Iterating Through 2D Arrays

===== Using Traditional For Loops

[source]
----
public class ZipCode {

  void compute() {
    int[][] matrix = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    
    // Print all elements
    for (int row = 0; row < matrix.length; row++) {
        for (int col = 0; col < matrix[row].length; col++) {
            System.out.print(matrix[row][col] + " ");
        }
        System.out.println(); // New line after each row
    }
  }

  public static void main(String[] args) { new ZipCode().compute(); }
}
----

===== Using Enhanced For Loops

[source]
----
public class ZipCode {

  void compute() {
    int[][] matrix = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };
    
    // Print all elements using enhanced for loops
    for (int[] row : matrix) {
        for (int element : row) {
            System.out.print(element + " ");
        }
        System.out.println();
    }
  }

  public static void main(String[] args) { new ZipCode().compute(); }
}
----

==== Practical Example: Grade Book

Here's a realistic example of using a 2D array to store student grades:

[source]
----
public class ZipCode {

  void compute() {
    // grades[student][test]
    // 4 students, 3 tests each
    int[][] grades = {
        {85, 92, 78},    // Student 0
        {90, 88, 95},    // Student 1
        {76, 89, 82},    // Student 2
        {88, 91, 85}     // Student 3
    };
    
    // Calculate and print each student's average
    for (int student = 0; student < grades.length; student++) {
        int total = 0;
        for (int test = 0; test < grades[student].length; test++) {
            total += grades[student][test];
        }
        double average = (double) total / grades[student].length;
        System.out.println("Student " + student + " average: " + average);
    }
    
    // Find the highest grade in all tests
    int highest = grades[0][0];
    for (int student = 0; student < grades.length; student++) {
        for (int test = 0; test < grades[student].length; test++) {
            if (grades[student][test] > highest) {
                highest = grades[student][test];
            }
        }
    }
    System.out.println("Highest grade: " + highest);
  }

  public static void main(String[] args) { new ZipCode().compute(); }
}
----

==== Understanding Array Dimensions

[source]
----
int[][] array2D = new int[3][4];  // 3 rows, 4 columns

// Getting dimensions:
int rows = array2D.length;           // 3
int columns = array2D[0].length;     // 4 (length of first row)

System.out.println("Rows: " + rows);
System.out.println("Columns: " + columns);
----

==== Jagged Arrays

In Java, you can have "jagged" arrays where each row has a different number of columns:

[source]
----
public class ZipCode {

  void compute() {
    // Create a jagged array
    int[][] jagged = {
        {1, 2},
        {3, 4, 5, 6},
        {7, 8, 9}
    };
    
    // Print the jagged array
    for (int row = 0; row < jagged.length; row++) {
        for (int col = 0; col < jagged[row].length; col++) {
            System.out.print(jagged[row][col] + " ");
        }
        System.out.println();
    }
  }

  public static void main(String[] args) { new ZipCode().compute(); }
}
----

==== 3D Arrays and Beyond

You can create arrays with more than 2 dimensions:

[source]
----
// A 3D array: [layers][rows][columns]
int[][][] cube = new int[2][3][4];

// Accessing a 3D element
cube[0][1][2] = 42;

// Getting dimensions
int layers = cube.length;           // 2
int rows = cube[0].length;          // 3  
int columns = cube[0][0].length;    // 4
----

==== Common Use Cases for 2D Arrays

1. **Game Boards** - Chess, checkers, tic-tac-toe
2. **Matrices** - Mathematical operations
3. **Images** - Pixel data (each element represents a pixel)
4. **Spreadsheets** - Rows and columns of data
5. **Seating Charts** - Rows and seats
6. **Maps** - Grid-based game maps
